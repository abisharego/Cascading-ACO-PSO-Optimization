# Cascading ACO-PSO Optimization for VM Scheduling in Cloud Computing

## Overview

This project implements a hybrid **Ant Colony Optimization (ACO)** and **Particle Swarm Optimization (PSO)** approach for Virtual Machine (VM) scheduling in cloud computing environments. The objective is to optimize resource allocation, reduce VM deployment latency, improve load balancing, and enhance overall system performance.

The proposed cascading framework utilizes ACO to identify promising scheduling solutions and PSO to further refine resource allocation decisions, resulting in efficient and scalable cloud resource management.

## System Workflow

1. Cloud workloads are generated.
2. ACO identifies promising VM-host mappings.
3. PSO refines the candidate solutions generated by ACO.
4. Optimized VM allocation is performed.
5. Performance metrics are evaluated and compared.
